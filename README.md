📌 Sobre o Projeto

A ideia central é um sistema onde usuários possam cadastrar doações e encontrar itens disponíveis de forma simples e organizada.


---

🚀 Tecnologias Utilizadas

Até o momento:

Java

Spring Boot

Maven

loombok



---

📂 Estrutura do Projeto

O projeto segue o padrão de arquitetura MVC (Model-View-Controller):

Model: Representação das entidades do sistema

Controller: Responsável por receber as requisições

Service: Contém as regras de negócio

Repository: Comunicação com o banco de dados


---

⚙️ Funcionalidades (em andamento)

Cadastro/Visualização e Exclusão de beneficiários
Cadastro/Visualização e Exclusão de doadores 
Cadastro/Visualização/Edição e Exclusão de Itens
